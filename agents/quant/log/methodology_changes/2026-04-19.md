# quant — 2026-04-19

Version: 1.26.0 -> 1.27.0

The current momentum coefficient of 0.00005 is far too conservative, resulting in predictions that barely deviate from current prices and terrible MSE performance (3.5x worse than naive baseline). I'm increasing the momentum coefficient from 0.00005 to 0.0003 to allow for more significant price movements based on the linear momentum signal. This should help capture actual price trends better and improve both MSE and directional accuracy. The reversion coefficient remains unchanged to maintain the mean-reversion component.
