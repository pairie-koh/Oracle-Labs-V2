# quant — 2026-04-07

Version: 1.18.0 -> 1.19.0

Reverting momentum calculation from EWMA back to linear regression slope. The EWMA approach appears to be generating too much noise in the momentum signal, contributing to poor directional accuracy. Linear regression provides a more stable trend estimate by fitting a line through recent price points rather than exponentially weighting recent observations. This should improve signal quality in the current stable market regime where small price fluctuations might be causing EWMA to overreact.
