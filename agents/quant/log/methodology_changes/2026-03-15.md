# Quant Methodology Change — 2026-03-15

**Version:** 1.1.0 → 1.2.0

## Change

Reduced the momentum forecast pull coefficient from `0.5` to `0.15`.

```python
# Before
momentum_forecast = current + (ewma - current) * 0.5

# After
momentum_forecast = current + (ewma - current) * 0.15
```

## Reasoning

Quant's overall MSE (0.00485) is ~60× worse than the naive baseline (0.00008). The market is in a low-volatility stable regime where prices barely move — the naive baseline wins by simply predicting the current price. The 0.5 coefficient caused the model to chase the EWMA aggressively whenever it lagged behind a recent price move (e.g., after a drop, EWMA sits above current, pulling the prediction 5–10pp higher than actual). Reducing the coefficient to 0.15 makes predictions stickier to current price while preserving a small directional lean from the EWMA signal. Directional accuracy is already excellent (0.9706 per market) — the problem is magnitude, not direction.
