# Methodology Change — 2026-03-19

**Version:** 1.5.0 → 1.6.0

**Change:** Reduced `ESCALATION_SENSITIVITY` from 0.0007 to 0.0004.

**Reasoning:** Per-market directional accuracy is 94.12% — the direction logic is working correctly. The remaining problem is magnitude: overall MSE is 0.00026561, 2.5× worse than the naive baseline of 0.00010589. The mse_trend_last_5 shows volatile recent cycles ([9.003e-05, 3.734e-05, 0.00023019]), indicating that even at 0.0007 sensitivity the agent still overshoots when signal strength is high (e.g., signal=10.65 this cycle). Reducing sensitivity 43% to 0.0004 shrinks shifts proportionally, keeping predictions tighter to the current price without disturbing direction, which continues the successful trajectory established across v1.2.0–v1.5.0.

**Expected effect:** Smaller per-cycle shifts should reduce squared error on high-signal cycles, bringing recent-cycle MSE consistently below the naive baseline of 0.000106. Direction is already correct; this change only reduces overshoot magnitude.
