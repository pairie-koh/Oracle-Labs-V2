# game_theorist — 2026-04-06

Version: 1.17.0 -> 1.18.0

I'm increasing ESCALATION_SENSITIVITY from 0.005 to 0.025 (5x increase) to make predictions more responsive to signal changes. The current sensitivity is so low that predictions barely move from current prices, leading to terrible directional accuracy and MSE performance. This change should allow the model to capture meaningful directional movements while still maintaining the game theory framework of costly signaling and actor incentives.
