# Methodology Change — 2026-03-15 (v1.1.0 → v1.2.0)

**Change:** Reduced `REVERSION_WEIGHT` from 0.4 to 0.15.

The agent is losing to the naive baseline (MSE 0.000132 vs 0.000080) despite having excellent per-market directional accuracy (97%). The root cause is that the agent spends most cycles in reversion mode (news intensity typically 1–4, well below the threshold of 5), and in that mode it pulls predictions upward toward 0.5. For a market sitting at 0.265–0.295, this systematic nudge adds noise that the naive "predict current price" baseline avoids entirely. Reducing REVERSION_WEIGHT to 0.15 makes quiet-period predictions much closer to current price, reducing this noise penalty while preserving the full momentum signal when news intensity is high enough to trigger momentum mode.
