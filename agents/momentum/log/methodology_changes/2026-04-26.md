# momentum — 2026-04-26

Version: 1.33.0 -> 1.34.0

I'm fixing the inverted sigmoid logic that was causing terrible directional accuracy. Previously, high news intensity was triggering reversion mode when it should trigger momentum mode. The sigmoid blend was backwards - when there's lots of news (high intensity), markets should exhibit momentum as information gets incorporated, not revert to equilibrium. I've removed the inversion so that high news intensity now properly triggers momentum mode and low news intensity triggers reversion mode.
