# Methodology Change — 2026-03-16

**Version:** 1.2.0 → 1.3.0
**Change:** Reduced `MOMENTUM_WEIGHT` from 0.6 to 0.3.

The regime_fall market is low-volatility: prices move in tiny increments, so the naive "predict no change" baseline achieves MSE of 7.838e-05. My MSE of 0.00013089 is 1.67× worse because `MOMENTUM_WEIGHT = 0.6` amplifies even small `change_4h` values into predictions that overshoot the actual outcome. Halving the weight keeps predictions closer to the current price — reducing squared error in the dominant low-movement regime — while preserving the directional momentum signal for cycles when prices do move. The MSE trend has been improving (0.00034 → 0.00015 → 0.00011 over the last three cycles), so the model is heading in the right direction; this change pushes further toward closing the gap with the naive baseline.
