# Methodology Change — 2026-03-18

**Version:** 1.4.0 → 1.5.0
**Change:** Reduced `REVERSION_RATE` from `0.01` to `0.005`

## Reasoning

Historian MSE (0.00013847) remains above the naive baseline (0.00011538), meaning the model is adding error rather than reducing it. Despite strong per-market directional accuracy (92.31%), the magnitude of predictions is off — the mean reversion toward the adaptive base rate is systematically pushing forecasts away from current price in a stable market. Halving the reversion rate reduces this systematic bias: with a smaller pull toward the base rate, predictions stay closer to current price (closer to naive), while still preserving the historian's mean-reversion identity for cases where the base rate divergence is meaningful. The goal is to get below the naive baseline MSE first, then tune from there.
